Cyto-bridge

Plugin to transfer data between Cytoscape and R, MATLAB, and other tools.

Download this project as a .zip file Download this project as a tar.gz file

Introduction

Cytoscape is a widely used bioinformatics software platform for integrating, analyzing, and visualizing biological network data sets. Cyto-bridge is a plugin that allows sharing of network data between other data-analysis programs such as R or Matlab and Cytoscape.

About Cyto-bridge

This is one of the Google Summer of Code 2012 projects.
